Important Concepts

How the web works

The web is a bunch of computers that communicate with each other. A web browser on your computer communicates with the server with HTTP. The server frinds the correct HTML document and sends it back to your browser. The computer interprets the page and displays it on your screen.

The web is a collection of HTML pages.

HTML

HTML stands for Hypertext Markup Language. It is the basis for everything on the web.

HTML is made of:
*text content = what you see
*Markup = what it looks like
*References to other docs = eg. images and videos
*Links to other pages

HTML Markups

< name> contents < / name>

Opening tags - < name>
Closing tags - < / name>

Common Markups

< b> = bold
< em> = italics
< a> = anchor is for making links
< img> = images
< br> = line breaks. Like the enter key when typing for a new line.

Inline vs Block

Inline are tags that are displayed without line breaks. Block tags have an invisible box around them, and normally start and end with a new line. The following are inline elements: a, br, em, b, img, span The following are block elements: p, div, and form.

Basic HTML vocab

Tag: contained with angled brackets. > <

Opening Tag: < content >

Closing Tag: < / content >

Void Tags: Tags that don't require closing tags, like < br> or < img>

Element: Everything within a set of opening and closing tags.

Attribute:A property of an HTML element.
